# AKBoard Engine (Node.js)

운영 가능한 수준의 인증/권한/차단/게시판 관리 기능을 제공하는 AKBoard 엔진입니다.

## 핵심 기능
- **최초 실행 설치 마법사** (`/install`)
  - 환영 문구 및 설치 단계 제공
  - 사이트 이름, 데이터베이스 생성 이름, 최고 관리자 아이디/비밀번호 설정
  - 설치 완료 후 "설치 후 환영합니다!" 단계 제공
- **회원가입 필수 입력 항목**
  - 학번
  - 이름(본명)
  - 닉네임
  - 아이디/비밀번호
  - `{사이트이름} 이용약관 동의` 체크
  - `개인정보 처리방침 동의` 체크
- **가입 마지막 단계 인증**
  - 이메일 가입: SMTP(`smtp.gmail.com`, `auth@parvia.kr`) 인증코드 검증 필수
  - 전화번호 가입: `1666-3538`로 랜덤 영숫자 코드 문자 발송 후 검증
  - 전화번호 검증은 OCTOMO API 검증 결과가 true일 때만 완료
- **Google/GitHub 연동**
  - 회원가입/로그인 화면에서 소셜 연동 진입
- **2FA 지원**
  - 전역 2FA 강제 설정 가능
  - 사용자별 2FA 활성/비활성 가능
- **아이디/비밀번호 찾기**
  - 아이디 찾기: 전화번호 문자 인증 기반
  - 미등록 시 학번/이름 지참 후 채널톡 문의 안내
  - 비밀번호 찾기: 이메일/전화번호 인증 후 비밀번호 초기화
- **어드민 패널 설정 가능 항목**
  - 약관/개인정보 동의 문구 수정
  - SMTP/앱 비밀번호/OCTOMO 인증 정보 수정
  - 유저 차단/권한/IP 차단/게시판 관리
- **안정성 강화**
  - 로그인/복구 경로 시도 제한(브루트포스 방지)
  - 관리자 주요 작업 감사 로그 기록
  - 전화번호/이메일 마스킹 표시
  - SMS 자동 발송 시도 + 수동 1666-3538 절차 병행

## 빠른 시작
```bash
npm install
npm start
```

기본 접속: `http://localhost:3000`

## 운영 설정
1. 서버 실행 후 `/install` 페이지에서 초기 설치 수행
2. 소셜 로그인 사용 시 Google/GitHub OAuth 앱 콜백 URL 설정
   - `http://localhost:3000/auth/google/callback`
   - `http://localhost:3000/auth/github/callback`
3. 어드민 패널(`/admin/dashboard`)에서 약관/인증/2FA 정책을 운영 중 변경
